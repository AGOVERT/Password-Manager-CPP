# Console Password Manager (C++)

Мой первый проект на C++


## 🚀 Особенности проекта
* **Безопасность:** Данные шифруются с использованием алгоритма XOR и кастомного мастер-пароля.
* **Архитектура:** (ООП) на C++, разделенный на логические модули (`.h` и `.cpp` файлы).
* **Интерфейс:** Удобное консольное меню с возможностью скрывать/показывать пароли при выводе на экран.

## 🛠️ Структура файлов проекта
* `spiski.cpp` — Главная точка входа в программу, инициализация и обработка основного меню.
* `PasswordManager.cpp` / `PasswordManager.h` — Логика управления хранилищем (чтение, запись, добавление записей, XOR-шифрование).
* `PasswordRecord.h` — Класс-инкапсуляция для одной записи аккаунта (сайт, логин, пароль).
* `spiski.sln` / `spiski.vcxproj` — Файлы конфигурации решения для Microsoft Visual Studio.

## 💻 Как запустить локально
1. Склонируйте данный репозиторий к себе на компьютер.
2. Откройте файл `spiski.sln` через **Microsoft Visual Studio**.
3. Убедитесь, что выбрана архитектура `x64` (`Debug` или `Release`).
4. Нажмите `F5` или кнопку **Локальный отладчик Windows** для компиляции и запуска программы.
